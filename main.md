# Интструкция лоя работы с Markdown

## Выделение текста

    Чтобы выделить текст курсивом необходимо обрамить его звёздочками (*).

Например _вот так_ .

    Чтобы выделить текст полкжирным необходимо обрамить двойными звёздочками (**).

Например **вот так**
